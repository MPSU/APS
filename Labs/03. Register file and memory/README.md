## –¢–µ–æ—Ä–∏—è –ø—Ä–æ –ø–∞–º—è—Ç—å

–ü–∞–º—è—Ç—å ‚Äî —ç—Ç–æ —É—Å—Ç—Ä–æ–π—Å—Ç–≤–æ –¥–ª—è —É–ø–æ—Ä—è–¥–æ—á–µ–Ω–Ω–æ–≥–æ —Ö—Ä–∞–Ω–µ–Ω–∏—è –∏ –≤—ã–¥–∞—á–∏ –∏–Ω—Ñ–æ—Ä–º–∞—Ü–∏–∏. –†–∞–∑–ª–∏—á–Ω—ã–µ –∑–∞–ø–æ–º–∏–Ω–∞—é—â–∏–µ —É—Å—Ç—Ä–æ–π—Å—Ç–≤–∞ –æ—Ç–ª–∏—á–∞—é—Ç—Å—è —Å–ø–æ—Å–æ–±–æ–º –∏ –æ—Ä–≥–∞–Ω–∏–∑–∞—Ü–∏–µ–π —Ö—Ä–∞–Ω–µ–Ω–∏—è –¥–∞–Ω–Ω—ã—Ö. –ë–∞–∑–æ–≤—ã–º–∏ —Ö–∞—Ä–∞–∫—Ç–µ—Ä–∏—Å—Ç–∏–∫–∞–º–∏ –ø–∞–º—è—Ç–∏ —è–≤–ª—è—é—Ç—Å—è:

- V ‚Äî –æ–±—ä–µ–º (–∫–æ–ª–∏—á–µ—Å—Ç–≤–æ –±–∏—Ç –¥–∞–Ω–Ω—ã—Ö, –∫–æ—Ç–æ—Ä—ã–µ –µ–¥–∏–Ω–æ–≤—Ä–µ–º–µ–Ω–Ω–æ –º–æ–∂–µ—Ç —Ö—Ä–∞–Ω–∏—Ç—å –ø–∞–º—è—Ç—å);
- a ‚Äî —Ä–∞–∑—Ä—è–¥–Ω–æ—Å—Ç—å –∞–¥—Ä–µ—Å–∞ (—à–∏—Ä–∏–Ω–∞ —à–∏–Ω—ã –∞–¥—Ä–µ—Å–∞, –æ–ø—Ä–µ–¥–µ–ª—è–µ—Ç –∞–¥—Ä–µ—Å–Ω–æ–µ –ø—Ä–æ—Å—Ç—Ä–∞–Ω—Å—Ç–≤–æ ‚Äî –∫–æ–ª–∏—á–µ—Å—Ç–≤–æ –∞–¥—Ä–µ—Å–æ–≤ –æ—Ç–¥–µ–ª—å–Ω—ã—Ö —è—á–µ–µ–∫ –ø–∞–º—è—Ç–∏);
- d ‚Äî —Ä–∞–∑—Ä—è–¥–Ω–æ—Å—Ç—å —Ö—Ä–∞–Ω–∏–º—ã—Ö –¥–∞–Ω–Ω—ã—Ö (—Ä–∞–∑—Ä—è–¥–Ω–æ—Å—Ç—å —è—á–µ–π–∫–∏ –ø–∞–º—è—Ç–∏, –∫–∞–∫ –ø—Ä–∞–≤–∏–ª–æ —Å–æ–≤–ø–∞–¥–∞–µ—Ç —Å —Ä–∞–∑—Ä—è–¥–Ω–æ—Å—Ç—å—é –≤—Ö–æ–¥–Ω—ã—Ö/–≤—ã—Ö–æ–¥–Ω—ã—Ö –¥–∞–Ω–Ω—ã—Ö).

–í –æ–±—â–µ–º —Å–ª—É—á–∞–µ `V = 2^a * d`.

–î–ª—è –æ–±—ä–µ–º–∞ –ø–∞–º—è—Ç–∏ –≤ 1 KiB ([–∫–∏–±–∏–±–∞–π—Ç](https://ru.wikipedia.org/wiki/%D0%9A%D0%B8%D0%B1%D0%B8%D0%B1%D0%B0%D0%B9%D1%82), 1024 –±–∞–π—Ç–∞ –∏–ª–∏ 8192 –±–∏—Ç–∞) —Ä–∞–∑—Ä—è–¥–Ω–æ—Å—Ç—å –∞–¥—Ä–µ—Å–∞ –º–æ–∂–µ—Ç –±—ã—Ç—å, –Ω–∞–ø—Ä–∏–º–µ—Ä, 10 –±–∏—Ç (—á—Ç–æ –ø–æ–∫—Ä—ã–≤–∞–µ—Ç 2^10 = 1024 –∞–¥—Ä–µ—Å–∞), —Ç–æ–≥–¥–∞ —Ä–∞–∑—Ä—è–¥–Ω–æ—Å—Ç—å —Ö—Ä–∞–Ω–∏–º—ã—Ö –¥–∞–Ω–Ω—ã—Ö –¥–æ–ª–∂–Ω–∞ –±—ã—Ç—å 8 –±–∏—Ç. 1024¬†*¬†8¬†=¬†8192, —Ç–æ –µ—Å—Ç—å 1¬†–∫–∏–±–∏–±–∞–π—Ç. –ï—Å–ª–∏ —Ä–∞–∑—Ä—è–¥–Ω–æ—Å—Ç—å –∞–¥—Ä–µ—Å–∞, –Ω–∞–ø—Ä–∏–º–µ—Ä, 8 –±–∏—Ç (—á—Ç–æ –ø–æ–∫—Ä—ã–≤–∞–µ—Ç 2^8 = 256 –∞–¥—Ä–µ—Å–æ–≤), —Ç–æ —Ä–∞–∑—Ä—è–¥–Ω–æ—Å—Ç—å –¥–∞–Ω–Ω—ã—Ö `d = V / 2^a` —ç—Ç–æ 8192¬†/¬†256 = 32 –±–∏—Ç–∞.

–û–¥–Ω–∞–∫–æ, –º–æ–∂–µ—Ç –±—ã—Ç—å —Ç–∞–∫–æ–µ, —á—Ç–æ –Ω–µ –≤—Å–µ —è—á–µ–π–∫–∏ –ø–∞–º—è—Ç–∏ —Ä–µ–∞–ª—å–Ω–æ —Ä–µ–∞–ª–∏–∑–æ–≤–∞–Ω—ã –Ω–∞ –∫—Ä–∏—Å—Ç–∞–ª–ª–µ –º–∏–∫—Ä–æ—Å—Ö–µ–º—ã, —Ç–æ –µ—Å—Ç—å –Ω–µ–∫–æ—Ç–æ—Ä—ã–µ –∞–¥—Ä–µ—Å–∞ —Å—É—â–µ—Å—Ç–≤—É—é—Ç, –Ω–æ –ø–æ –Ω–∏–º –Ω–µ –∏–º–µ–µ—Ç —Å–º—ã—Å–ª–∞ –æ–±—Ä–∞—â–∞—Ç—å—Å—è, –∞ –æ–±—ä–µ–º –ø–∞–º—è—Ç–∏, —Å–æ–æ—Ç–≤–µ—Ç—Å—Ç–≤–µ–Ω–Ω–æ, –Ω–µ —Ä–∞–≤–µ–Ω `V ‚â† 2^a * d` ‚Äî –æ–Ω –º–µ–Ω—å—à–µ. –ü–æ–¥–æ–±–Ω—ã–µ —Å–ª—É—á–∞–∏ –±—É–¥—É—Ç —Ä–∞—Å—Å–º–æ—Ç—Ä–µ–Ω—ã –æ—Ç–¥–µ–ª—å–Ω–æ.

–ü–∞–º—è—Ç—å –º–æ–∂–Ω–æ —Ä–∞–∑–¥–µ–ª–∏—Ç—å –Ω–∞ –∫–∞—Ç–µ–≥–æ—Ä–∏–∏: –ü–ó–£ (–ø–æ—Å—Ç–æ—è–Ω–Ω–æ–µ –∑–∞–ø–æ–º–∏–Ω–∞—é—â–µ–µ —É—Å—Ç—Ä–æ–π—Å—Ç–≤–æ) –∏ –û–ó–£ (–æ–ø–µ—Ä–∞—Ç–∏–≤–Ω–æ–µ –∑–∞–ø–æ–º–∏–Ω–∞—é—â–µ–µ —É—Å—Ç—Ä–æ–π—Å—Ç–≤–æ). –ò–∑ –ü–ó–£ –º–æ–∂–Ω–æ —Ç–æ–ª—å–∫–æ —Å—á–∏—Ç—ã–≤–∞—Ç—å –∏–Ω—Ñ–æ—Ä–º–∞—Ü–∏—é, –∫–æ—Ç–æ—Ä–∞—è –ø–æ–ø–∞–¥–∞–µ—Ç –≤ –ü–ó–£ –¥–æ –Ω–∞—á–∞–ª–∞ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏—è –ø–∞–º—è—Ç–∏ –∏ –Ω–µ –º–æ–∂–µ—Ç –∏–∑–º–µ–Ω—è—Ç—å—Å—è –≤ –ø—Ä–æ—Ü–µ—Å—Å–µ —Ä–∞–±–æ—Ç—ã. –ò–∑ –û–ó–£ –º–æ–∂–Ω–æ —Å—á–∏—Ç—ã–≤–∞—Ç—å –∏ –∑–∞–ø–∏—Å—ã–≤–∞—Ç—å –∏–Ω—Ñ–æ—Ä–º–∞—Ü–∏—é. –í —Å–∞–º–æ–º –ø—Ä–æ—Å—Ç–æ–º —Å–ª—É—á–∞–µ –ü–ó–£ –∏–º–µ–µ—Ç –æ–¥–∏–Ω –≤—Ö–æ–¥ –∞–¥—Ä–µ—Å–∞ `addr` –∏ –æ–¥–∏–Ω –≤—ã—Ö–æ–¥ —Å—á–∏—Ç—ã–≤–∞–µ–º—ã—Ö –¥–∞–Ω–Ω—ã—Ö `read_data`. –ù–∞ –≤—Ö–æ–¥ `addr` –ø–æ–¥–∞–µ—Ç—Å—è –∞–¥—Ä–µ—Å —Ç—Ä–µ–±—É–µ–º–æ–π —è—á–µ–π–∫–∏ –ø–∞–º—è—Ç–∏, –Ω–∞ –≤—ã—Ö–æ–¥–µ `read_data` –ø–æ—è–≤–ª—è—é—Ç—Å—è –¥–∞–Ω–Ω—ã–µ, –∫–æ—Ç–æ—Ä—ã–µ —Ö—Ä–∞–Ω—è—Ç—Å—è –ø–æ —ç—Ç–æ–º—É –∞–¥—Ä–µ—Å—É.

–î–ª—è –û–ó–£ —Ç—Ä–µ–±—É–µ—Ç—Å—è –±–æ–ª—å—à–µ —Å–∏–≥–Ω–∞–ª–æ–≤. –ö—Ä–æ–º–µ –≤—Ö–æ–¥–Ω–æ–≥–æ `addr` –∏ –≤—ã—Ö–æ–¥–Ω–æ–≥–æ `read_data` –¥–æ–±–∞–≤–ª—è—é—Ç—Å—è: –≤—Ö–æ–¥–Ω—ã–µ –¥–∞–Ω–Ω—ã–µ –¥–ª—è –∑–∞–ø–∏—Å–∏ `write_data`, —Å–∏–≥–Ω–∞–ª —Å–∏–Ω—Ö—Ä–æ–Ω–∏–∑–∞—Ü–∏–∏ `clk`, –∫–æ—Ç–æ—Ä—ã–π –æ–ø—Ä–µ–¥–µ–ª—è–µ—Ç –º–æ–º–µ–Ω—Ç –∑–∞–ø–∏—Å–∏ –¥–∞–Ω–Ω—ã—Ö –∏ —Å–∏–≥–Ω–∞–ª —Ä–∞–∑—Ä–µ—à–µ–Ω–∏—è –Ω–∞ –∑–∞–ø–∏—Å—å `write_enable`, –∫–æ—Ç–æ—Ä—ã–π –∫–æ–Ω—Ç—Ä–æ–ª–∏—Ä—É–µ—Ç –Ω—É–∂–Ω–æ –ª–∏ –∑–∞–ø–∏—Å—ã–≤–∞—Ç—å –¥–∞–Ω–Ω—ã–µ –∏–ª–∏ —Ç–æ–ª—å–∫–æ —Å—á–∏—Ç—ã–≤–∞—Ç—å. –î–ª—è —Ç–æ–≥–æ, —á—Ç–æ–±—ã –∑–∞–ø–∏—Å–∞—Ç—å –∏–Ω—Ñ–æ—Ä–º–∞—Ü–∏—é –≤ —Ç–∞–∫—É—é –ø–∞–º—è—Ç—å –Ω–µ–æ–±—Ö–æ–¥–∏–º–æ:

- –≤—ã—Å—Ç–∞–≤–∏—Ç—å –∞–¥—Ä–µ—Å `addr` –≤ –∫–æ—Ç–æ—Ä—ã–π –ø–ª–∞–Ω–∏—Ä—É–µ—Ç—Å—è –∑–∞–ø–∏—Å—å –¥–∞–Ω–Ω—ã—Ö,
- –≤—ã—Å—Ç–∞–≤–∏—Ç—å —Å–∞–º–∏ –¥–∞–Ω–Ω—ã–µ –¥–ª—è –∑–∞–ø–∏—Å–∏ –Ω–∞ –≤—Ö–æ–¥ `write_data`,
- —É—Å—Ç–∞–Ω–æ–≤–∏—Ç—å —Å–∏–≥–Ω–∞–ª `write_enable` –≤ —Å–æ—Å—Ç–æ—è–Ω–∏–µ —Ä–∞–∑—Ä–µ—à–µ–Ω–∏—è –∑–∞–ø–∏—Å–∏ (–∫–∞–∫ –ø—Ä–∞–≤–∏–ª–æ —ç—Ç–æ 1) –∏
- –¥–æ–∂–¥–∞—Ç—å—Å—è –Ω—É–∂–Ω–æ–≥–æ —Ñ—Ä–æ–Ω—Ç–∞ `clk` ‚Äî –≤ —ç—Ç–æ—Ç –º–æ–º–µ–Ω—Ç –¥–∞–Ω–Ω—ã–µ –±—É–¥—É—Ç –∑–∞–ø–∏—Å–∞–Ω—ã –ø–æ —É–∫–∞–∑–∞–Ω–Ω–æ–º—É –∞–¥—Ä–µ—Å—É. –ü—Ä–∏ —ç—Ç–æ–º, –Ω–∞ –≤—ã—Ö–æ–¥–µ `read_data` –±—É–¥—É—Ç —Å—Ç–∞—Ä—ã–µ –¥–∞–Ω–Ω—ã–µ, —Ö—Ä–∞–Ω—è—â–∏–µ—Å—è –ø–æ –∞–¥—Ä–µ—Å—É `addr`. –ù–∞ –æ–¥–Ω–æ–º —Ç–∞–∫—Ç–µ –ø—Ä–æ–∏—Å—Ö–æ–¥–∏—Ç –æ–¥–Ω–æ–≤—Ä–µ–º–µ–Ω–Ω–æ–µ —Å—á–∏—Ç—ã–≤–∞–Ω–∏–µ –∏–Ω—Ñ–æ—Ä–º–∞—Ü–∏–∏ –∏ –∑–∞–ø–∏—Å—å –Ω–æ–≤–æ–π.

–¢–∞–∫ –∂–µ –≤–æ–∑–º–æ–∂–Ω–∞ —Ä–µ–∞–ª–∏–∑–∞—Ü–∏—è, –≤ –∫–æ—Ç–æ—Ä–æ–π –≤—Ö–æ–¥ `read_data` –∏ –≤—ã—Ö–æ–¥ `write_data` –æ–±—ä–µ–¥–∏–Ω–µ–Ω—ã –≤ –µ–¥–∏–Ω—ã–π –≤—Ö–æ–¥/–≤—ã—Ö–æ–¥ `data`. –í —ç—Ç–æ–º —Å–ª—É—á–∞–µ –æ–ø–µ—Ä–∞—Ü–∏–∏ —á—Ç–µ–Ω–∏—è –∏ –∑–∞–ø–∏—Å–∏ —Ä–∞–∑–¥–µ–ª–µ–Ω—ã –≤–æ –≤—Ä–µ–º–µ–Ω–∏ –∏ –∏—Å–ø–æ–ª—å–∑—É—é—Ç –¥–ª—è —ç—Ç–æ–≥–æ –æ–¥–∏–Ω –µ–¥–∏–Ω—ã–π –ø–æ—Ä—Ç –≤–≤–æ–¥–∞-–≤—ã–≤–æ–¥–∞ (`inout`, –¥–≤—É–Ω–∞–ø—Ä–∞–≤–ª–µ–Ω–Ω—ã–π –ø–æ—Ä—Ç) `data`.

–†–µ–∞–ª–∏–∑–∞—Ü–∏—è –∞—Å–∏–Ω—Ö—Ä–æ–Ω–Ω–æ–≥–æ –ø–æ–¥–∫–ª—é—á–µ–Ω–∏—è –∫ –≤—ã—Ö–æ–¥—É –ø–∞–º—è—Ç–∏ –æ—Å—É—â–µ—Å—Ç–≤–ª—è–µ—Ç—Å—è –æ–ø–µ—Ä–∞—Ç–æ—Ä–æ–º `assign`. –ê, –µ—Å–ª–∏ —Ç—Ä–µ–±—É–µ—Ç—Å—è —Å–æ–∑–¥–∞—Ç—å –ø–∞–º—è—Ç—å —Å —Å–∏–Ω—Ö—Ä–æ–Ω–Ω—ã–º —á—Ç–µ–Ω–∏–µ–º, —Ç–æ –ø—Ä–∏—Å–≤–∞–∏–≤–∞–Ω–∏–µ –≤—ã—Ö–æ–¥—É —Ç—Ä–µ–±—É–µ—Ç—Å—è –æ–ø–∏—Å–∞—Ç—å –≤–Ω—É—Ç—Ä–∏ –±–ª–æ–∫–∞`always_ff`.

–¢–∞–∫ –∫–∞–∫ –∑–∞–ø–∏—Å—å –≤ –ø–∞–º—è—Ç—å —è–≤–ª—è–µ—Ç—Å—è —Å–∏–Ω—Ö—Ä–æ–Ω–Ω—ã–º —Å–æ–±—ã—Ç–∏–µ–º, —Ç–æ –æ–ø–∏—Å—ã–≤–∞–µ—Ç—Å—è –æ–Ω–∞ –≤ –∫–æ–Ω—Å—Ç—Ä—É–∫—Ü–∏–∏ `always_ff`. –ü—Ä–∏ —ç—Ç–æ–º, –∫–∞–∫ –∏ –ø—Ä–∏ –æ–ø–∏—Å–∞–Ω–∏–∏ —Ä–µ–≥–∏—Å—Ç—Ä–∞, –º–æ–∂–Ω–æ —Ä–µ–∞–ª–∏–∑–æ–≤–∞—Ç—å —É–ø—Ä–∞–≤–ª—è—é—â–∏–π —Å–∏–≥–Ω–∞–ª —Ä–∞–∑—Ä–µ—à–µ–Ω–∏—è –Ω–∞ –∑–∞–ø–∏—Å—å —á–µ—Ä–µ–∑ –±–ª–æ–∫ –≤–∏–¥–∞ `if(write_enable)`.

### 1. –ü–∞–º—è—Ç—å –∏–Ω—Å—Ç—Ä—É–∫—Ü–∏–π

–ò–∑ [—Ç–µ–æ—Ä–∏–∏ –ø—Ä–æ –ø–∞–º—è—Ç—å](#—Ç–µ–æ—Ä–∏—è-–ø—Ä–æ-–ø–∞–º—è—Ç—å) –≤—ã –º–æ–≥–ª–∏ –¥–æ–≥–∞–¥–∞—Ç—å—Å—è, —á—Ç–æ —Ç–∞–∫–æ–π –º–æ–¥—É–ª—å –æ–ø–∏—Å—ã–≤–∞–µ—Ç –ø–∞–º—è—Ç—å —ë–º–∫–æ—Å—Ç—å—é `4 GiB`. –û–¥–Ω–∞–∫–æ –≤ —Ä–µ–∞–ª—å–Ω–æ—Å—Ç–∏, –Ω–∞—à–∞ –ø–∞–º—è—Ç—å –±—É–¥–µ—Ç –∫—É–¥–∞ –º–µ–Ω—å—à–µ (–≤ –ü–õ–ò–° –ø–æ–ø—Ä–æ—Å—Ç—É –Ω–µ —Ö–≤–∞—Ç–∏—Ç —Ä–µ—Å—É—Ä—Å–æ–≤ –Ω–∞ —Ä–µ–∞–ª–∏–∑–∞—Ü–∏—é –ø–∞–º—è—Ç–∏ –ø–æ–¥–æ–±–Ω–æ–≥–æ –æ–±—ä—ë–º–∞). –ù–∞ –ø—Ä–∞–∫—Ç–∏–∫–µ, –≤–Ω—É—Ç—Ä–∏ –¥–∞–Ω–Ω–æ–≥–æ –º–æ–¥—É–ª—è –≤—ã –¥–æ–ª–∂–Ω—ã –±—É–¥–µ—Ç–µ —Ä–µ–∞–ª–∏–∑–æ–≤–∞—Ç—å –ø–∞–º—è—Ç—å —Å 1024-–º—è 32-–±–∏—Ç–Ω—ã–º–∏ —è—á–µ–π–∫–∞–º–∏.

–ü—Ä–∏ —ç—Ç–æ–º –ø–æ —Å–ø–µ—Ü–∏—Ñ–∏–∫–∞—Ü–∏–∏ –ø—Ä–æ—Ü–µ—Å—Å–æ—Ä RISC-V –∏—Å–ø–æ–ª—å–∑—É–µ—Ç –ø–∞–º—è—Ç—å —Å –ø–æ–±–∞–π—Ç–æ–≤–æ–π –∞–¥—Ä–µ—Å–∞—Ü–∏–µ–π. –ë–∞–π—Ç–æ–≤–∞—è –∞–¥—Ä–µ—Å–∞—Ü–∏—è –æ–∑–Ω–∞—á–∞–µ—Ç, —á—Ç–æ –ø—Ä–æ—Ü–µ—Å—Å–æ—Ä —Å–ø–æ—Å–æ–±–µ–Ω –æ–±—Ä–∞—â–∞—Ç—å—Å—è –∫ –æ—Ç–¥–µ–ª—å–Ω—ã–º –±–∞–π—Ç–∞–º –≤ –ø–∞–º—è—Ç–∏ (–∑–∞ –∫–∞–∂–¥—ã–º –±–∞–π—Ç–æ–º –ø–∞–º—è—Ç–∏ –∑–∞–∫—Ä–µ–ø–ª–µ–Ω —Å–≤–æ–π –∏–Ω–¥–∏–≤–∏–¥—É–∞–ª—å–Ω—ã–π –∞–¥—Ä–µ—Å).

–û–¥–Ω–∞–∫–æ, –µ—Å–ª–∏ —É –ø–∞–º—è—Ç–∏ –±—É–¥—É—Ç 32-—Ä—è–∑—Ä—è–¥–Ω—ã–µ —è—á–µ–π–∫–∏, –¥–æ—Å—Ç—É–ø –∫ –∫–æ–Ω–∫—Ä–µ—Ç–Ω–æ–º—É –±–∞–π—Ç—É –±—É–¥–µ—Ç –æ—Å–ª–æ–∂–Ω–µ–Ω, –≤–µ–¥—å –∫–∞–∂–¥–∞—è —è—á–µ–π–∫–∞ ‚Äî —ç—Ç–æ 4 –±–∞–π—Ç–∞. –ö–∞–∫ –ø–æ–ª—É—á–∏—Ç—å –¥–∞–Ω–Ω—ã–µ —Ç—Ä–µ—Ç—å–µ–≥–æ –±–∞–π—Ç–∞ –ø–∞–º—è—Ç–∏? –ï—Å–ª–∏ –æ–±—Ä–∞—Ç–∏—Ç—å—Å—è –∫ —Ç—Ä–µ—Ç—å–µ–π —è—á–µ–π–∫–µ –≤ –º–∞—Å—Å–∏–≤–µ ‚Äî –ø—Ä–∏–¥—É—Ç –¥–∞–Ω–Ω—ã–µ 12-15-—ã—Ö –±–∞–π—Ç –±–∞–π—Ç (–ø–æ—Å–∫–æ–ª—å–∫—É –∫–∞–∂–¥–∞—è —è—á–µ–π–∫–∞ —Å–æ–¥–µ—Ä–∂–∏—Ç –ø–æ 4 –±–∞–π—Ç–∞). –ß—Ç–æ–±—ã –ø–æ–ª—É—á–∏—Ç—å –¥–∞–Ω–Ω—ã–µ —Ç—Ä–µ—Ç—å–µ–≥–æ –±–∞–π—Ç–∞, –Ω–µ–æ–±—Ö–æ–¥–∏–º–æ —Ä–∞–∑–¥–µ–ª–∏—Ç—å –ø—Ä–∏—à–µ–¥—à–∏–π –∞–¥—Ä–µ—Å –Ω–∞ 4 (–æ—Ç–±—Ä–æ—Å–∏–≤ –æ—Å—Ç–∞—Ç–æ–∫ –æ—Ç –¥–µ–ª–µ–Ω–∏—è). `3 / 4 = 0` ‚Äî –∏ –¥–µ–π—Å—Ç–≤–∏—Ç–µ–ª—å–Ω–æ, –µ—Å–ª–∏ –æ–±—Ä–∞—Ç–∏—Ç—å—Å—è –∫ –Ω—É–ª–µ–≤–æ–π —è—á–µ–π–∫–µ –ø–∞–º—è—Ç–∏ ‚Äî –±—É–¥—É—Ç –ø–æ–ª—É—á–µ–Ω—ã –¥–∞–Ω–Ω—ã–µ 3-–≥–æ, 2-–≥–æ, 1-–≥–æ –∏ 0-–≥–æ –±–∞–π—Ç. –¢–æ —á—Ç–æ –ø–æ–º–∏–º–æ –∑–Ω–∞—á–µ–Ω–∏—è —Ç—Ä–µ—Ç—å–µ–≥–æ –±–∞–π—Ç–∞ –µ—Å—Ç—å –µ—â–µ –¥–∞–Ω–Ω—ã–µ –¥—Ä—É–≥–∏—Ö –±–∞–π—Ç –Ω–∞—Å –≤ –¥–∞–Ω–Ω—ã–π –º–æ–º–µ–Ω—Ç –Ω–µ –∏–Ω—Ç–µ—Ä–µ—Å—É–µ—Ç, –≤–∞–∂–Ω–∞ —Ç–æ–ª—å–∫–æ —Å–∞–º–∞ –≤–æ–∑–º–æ–∂–Ω–æ—Å—Ç—å —É–∫–∞–∑–∞—Ç—å –∞–¥—Ä–µ—Å –∫–æ–Ω–∫—Ä–µ—Ç–Ω–æ–≥–æ –±–∞–π—Ç–∞.

–î–µ–ª–µ–Ω–∏–µ –Ω–∞ `2<sup>n</sup>` –º–æ–∂–Ω–æ –æ—Å—É—â–µ—Å—Ç–≤–∏—Ç—å –æ—Ç–±—Ä–æ—Å–∏–≤ `n` –º–ª–∞–¥—à–∏—Ö –±–∏—Ç —á–∏—Å–ª–∞. –¢–∞–∫–∏–º –æ–±—Ä–∞–∑–æ–º –Ω–∞ –≤—ã—Ö–æ–¥ –ø–∞–º—è—Ç—å –∏–Ω—Å—Ç—Ä—É–∫—Ü–∏–π –¥–æ–ª–∂–Ω–∞ –≤—ã–¥–∞–≤–∞—Ç—å –¥–∞–Ω–Ω—ã–µ, —Ä–∞—Å–ø–æ–ª–æ–∂–µ–Ω–Ω—ã–µ –ø–æ –∞–¥—Ä–µ—Å—É addr_i[31:2];

–û–±—Ä–∞—â–µ–Ω–∏–µ –≤ –ø–∞–º—è—Ç—å –ø–æ –∞–¥—Ä–µ—Å–∞–º, –ø—Ä–µ–≤—ã—à–∞—é—â–∏–º `4095` –¥–æ–ª–∂–Ω–æ –≤—ã–¥–∞–≤–∞—Ç—å –∑–Ω–∞—á–µ–Ω–∏–µ `32'd0`. –ü–æ—á–µ–º—É –∏–º–µ–Ω–Ω–æ `4095`? `4095 / 4 = 1023` ‚Äî –∏–Ω–¥–µ–∫—Å –ø–æ—Å–ª–µ–¥–Ω–µ–π —è—á–µ–π–∫–∏ –ø–∞–º—è—Ç–∏.

–ö–∞–∫ —Ä–µ–∞–ª–∏–∑–æ–≤–∞—Ç—å –ø–æ–¥–æ–±–Ω—ã–π —Ñ—É–Ω–∫—Ü–∏–æ–Ω–∞–ª? –†–∞–∑—É–º–µ–µ—Ç—Å—è, —Å –ø–æ–º–æ—â—å—é [–º—É–ª—å—Ç–∏–ø–ª–µ–∫—Å–æ—Ä–∞](../../Basic%20Verilog%20structures/Multiplexors.md).

–í –¥–∞–Ω–Ω–æ–º —Å–ª—É—á–∞–µ, –≤—ã –º–æ–∂–µ—Ç–µ –ª–∏–±–æ –≤–æ—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å—Å—è –∫–æ–Ω—Å—Ç—Ä—É–∫—Ü–∏–µ–π `if-else` –≤ –±–ª–æ–∫–µ `always_comb`, –ª–∏–±–æ **—Ç–µ—Ä–Ω–∞—Ä–Ω—ã–º —É—Å–ª–æ–≤–Ω—ã–º –æ–ø–µ—Ä–∞—Ç–æ—Ä–æ–º** –≤–æ –≤—Ä–µ–º—è —Ä–µ–∞–ª–∏–∑–∞—Ü–∏–∏ **–Ω–µ–ø—Ä–µ—Ä—ã–≤–Ω–æ–≥–æ –ø—Ä–∏—Å–≤–∞–∏–≤–∞–Ω–∏—è**. –í –ª—é–±–æ–º –∏–∑ —Å–ª—É—á–∞–µ–≤, –≤–∞–º –Ω—É–∂–Ω–æ –±—É–¥–µ—Ç —Ä–µ–∞–ª–∏–∑–æ–≤–∞—Ç—å —É—Å–ª–æ–≤–∏–µ —Ç–æ–≥–æ, —á—Ç–æ –ø—Ä–∏—à–µ–¥—à–µ–µ –∑–Ω–∞—á–µ–Ω–∏–µ –Ω–∞ —à–∏–Ω–µ –∞–¥—Ä–µ—Å–∞ –ø–æ–ø–∞–¥–∞–µ—Ç –≤ –¥–∏–∞–ø–∞–∑–æ–Ω –¥–æ–ø—É—Å—Ç–∏–º—ã—Ö –∞–¥—Ä–µ—Å–æ–≤. –ï—Å–ª–∏ —ç—Ç–æ —Ç–∞–∫, –≤–µ—Ä–Ω—É—Ç—å —Å–∫–ª–µ–π–∫—É –¥–∞–Ω–Ω—ã—Ö –∏–∑ –ø–∞–º—è—Ç–∏, –Ω–∞—á–∏–Ω–∞—é—â–∏—Ö—Å—è —Å —ç—Ç–æ–≥–æ –∞–¥—Ä–µ—Å–∞, –≤ –ø—Ä–æ—Ç–∏–≤–Ω–æ–º —Å–ª—É—á–∞–µ ‚Äî –≤–µ—Ä–Ω—É—Ç—å 0.

### 2. –ü–∞–º—è—Ç—å –¥–∞–Ω–Ω—ã—Ö

–£ –¥–∞–Ω–Ω–æ–π –ø–∞–º—è—Ç–∏ –±—É–¥–µ—Ç 8 –ø–æ—Ä—Ç–æ–≤:

- –≤—Ö–æ–¥ —Ç–∞–∫—Ç–æ–≤–æ–≥–æ —Å–∏–Ω—Ö—Ä–æ–∏–º–ø—É–ª—å—Å–∞
- –≤—Ö–æ–¥ –∑–∞–ø—Ä–æ—Å–∞ –Ω–∞ —Ä–∞–±–æ—Ç—É —Å –ø–∞–º—è—Ç—å—é
- –≤—Ö–æ–¥ —Å–∏–≥–Ω–∞–ª–∞ —Ä–∞–∑—Ä–µ—à–µ–Ω–∏—è –∑–∞–ø–∏—Å–∏
- 32-–±–∏—Ç–Ω—ã–π –≤—Ö–æ–¥ –∞–¥—Ä–µ—Å–∞
- 32-–±–∏—Ç–Ω—ã–π –≤—Ö–æ–¥ –¥–∞–Ω–Ω—ã—Ö –∑–∞–ø–∏—Å–∏
- 32-–±–∏—Ç–Ω—ã–π –≤—ã—Ö–æ–¥ –¥–∞–Ω–Ω—ã—Ö —Å–∏–Ω—Ö—Ä–æ–Ω–Ω–æ–≥–æ —á—Ç–µ–Ω–∏—è

```SystemVerilog
m–ædul–µ data_m–µm(
  in—Äut  logic        clk_i,
  input  logic        mem_req_i,
  in—Äut  logic        write_enable_i,
  in—Äut  logic [31:0] addr_i,
  in—Äut  logic [31:0] write_data_i,
  –æut—Äut logic [31:0] r–µ–∞d_d–∞t–∞_o

```

–ö–∞–∫ –∏ –ø–∞–º—è—Ç—å –∏–Ω—Å—Ç—Ä—É–∫—Ü–∏–π, –ø–∞–º—è—Ç—å –¥–∞–Ω–Ω—ã—Ö –±—É–¥–µ—Ç —Å–æ—Å—Ç–æ—è—Ç—å –∏–∑ 32-—Ä–∞–∑—Ä—è–¥–Ω—ã—Ö —è—á–µ–µ–∫ (—Ç–æ–ª—å–∫–æ —Ç–µ–ø–µ—Ä—å –∏—Ö –±—É–¥–µ—Ç 4096), –∏ –ø—Ä–∏ –æ–±—Ä–∞—â–µ–Ω–∏–∏ –∫ —ç—Ç–∏–º —è—á–µ–π–∫–∞–º –±—É–¥–µ—Ç –Ω–µ–æ–±—Ö–æ–¥–∏–º–æ –¥–µ–ª–∏—Ç—å –∞–¥—Ä–µ—Å –Ω–∞ 4.

–û–¥–Ω–∞–∫–æ –≤ –æ—Ç–ª–∏—á–∏–µ –æ—Ç –ø–∞–º—è—Ç–∏ –∏–Ω—Å—Ç—Ä—É–∫—Ü–∏–π, –≤ –ø–∞–º—è—Ç—å –¥–∞–Ω–Ω—ã—Ö –¥–æ–±–∞–≤–ª–µ–Ω–æ –¥–≤–∞ —É–ø—Ä–∞–≤–ª—è—é—â–∏—Ö —Å–∏–≥–Ω–∞–ª–∞ (`mem_req_i`–∏ `write_enable_i`). –°–∏–≥–Ω–∞–ª `mem_req_i` —è–≤–ª—è–µ—Ç—Å—è —Å–∏–≥–Ω–∞–ª–æ–º –∑–∞–ø—Ä–æ—Å–∞ –Ω–∞ —Ä–∞–±–æ—Ç—É —Å –ø–∞–º—è—Ç—å—é. –ë–µ–∑ —ç—Ç–æ–≥–æ —Å–∏–≥–Ω–∞–ª–∞ –ø–∞–º—è—Ç—å –Ω–µ –¥–æ–ª–∂–Ω–∞ –≤—ã–ø–æ–ª–Ω—è—Ç—å –æ–ø–µ—Ä–∞—Ü–∏–∏ —á—Ç–µ–Ω–∏—è/–∑–∞–ø–∏—Å–∏ (–≤–Ω–µ –∑–∞–≤–∏—Å–∏–º–æ—Å—Ç–∏ –æ—Ç —Å–∏–≥–Ω–∞–ª–∞ `write_enable`, –æ–ø—Ä–µ–¥–µ–ª—è—é—â–µ–≥–æ –ø—Ä–æ–∏—Å—Ö–æ–¥–∏—Ç —Å–µ–π—á–∞—Å –∑–∞–ø–∏—Å—å –∏–ª–∏ —á—Ç–µ–Ω–∏–µ). –ö–∞–∫ —Å–¥–µ–ª–∞—Ç—å —Ç–∞–∫, —á—Ç–æ–±—ã –Ω–µ –ø—Ä–æ–∏—Å—Ö–æ–¥–∏–ª–æ —á—Ç–µ–Ω–∏–µ –±–µ–∑ –∑–∞–ø—Ä–æ—Å–∞? –ù–∞–ø—Ä–∏–º–µ—Ä –≤–æ–∑–≤—Ä–∞—â–∞—Ç—å –Ω–∞ —à–∏–Ω—É —á—Ç–µ–Ω–∏—è —Å–ø–µ—Ü–∏–∞–ª—å–Ω–æ–µ "–º–∞–≥–∏—á–µ—Å–∫–æ–µ —á–∏—Å–ª–æ".

- –í —Å–ª—É—á–∞–µ `mem_req_i == 0` –∏–ª–∏ `write_enable_i == 1` (—Ç.–µ. –∫–æ–≥–¥–∞ –Ω–µ –≤—ã–ø–æ–ª–Ω—è–µ—Ç—Å—è –æ–ø–µ—Ä–∞—Ü–∏—è —á—Ç–µ–Ω–∏—è), –Ω–∞ –≤—ã—Ö–æ–¥–µ `read_data_o` –¥–æ–ª–∂–Ω–æ –æ–∫–∞–∑–∞—Ç—å—Å—è –∑–Ω–∞—á–µ–Ω–∏–µ `32'hfa11_1eaf` (–ø–æ—Å–∫–æ–ª—å–∫—É `1` —Å—Ö–æ–∂–∞ —Å –ª–∞—Ç–∏–Ω—Å–∫–∏–º —Å–∏–º–≤–æ–ª–æ–º `l`, —ç—Ç–æ –≤—ã—Ä–∞–∂–µ–Ω–∏–µ –º–æ–∂–Ω–æ –ø—Ä–æ—á–µ—Å—Ç—å –∫–∞–∫ `fall_leaf`).
